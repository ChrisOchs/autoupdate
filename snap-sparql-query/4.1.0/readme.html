Snap SPARQL Query

A plug-in for querying ontologies with SPARQL-DL. This plug-in was developed for tutorial purposes as part of the Protege Short Course series. We make no quality guarantees regarding use in production environments.

Version info

v1.0.0 (17th Mar 2014)

  • Initial release
  • Autoupdate activated

v2.0.0 (26th Mar 2015)

  • Implementation of some builtins

v3.0.0 (28th Sep 2015)

  • Implemented more builtins
  • Queries involving Optional are now supported
  • Aggregate queries are now supported
  • Construct queries are now supported
v4.0.0 (20th Jan 2016)
  • Upgraded to OWL API version 4.1.3
v4.1.0 (18th Mar 2016)
  • Fixed rendering and highlighting problems that could occur on Windows
  • Added a sample query on startup
